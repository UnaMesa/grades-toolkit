## Installation

Install [Meteorite](https://github.com/oortcloud/meteorite) and [Atmosphere](https://atmosphere.meteor.com)


  ```sh
  $ sudo -H npm install -g meteorite
  $ mrt update
  ```


## Quick Start

Run

```sh
$ mrt
```

You may have to enter credentials for google authorization.
